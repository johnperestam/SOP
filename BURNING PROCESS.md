### <a name="introduction"></a>**1. Introduction**
<ul><li>
	Overview of required procedures to write up and process sales order with burning operations.
</li></ul>

### **<a name="purpose"></a>3. Purpose**
<ul><li>
	Provide the proper procedures for entering and processing burning orders.
</li><li>
	Create consistency in order entry, inspection and packaging.
</li><li>
	Ensure employee safety and efficiency in completing daily tasks.
</li></ul>

### **<a name="scope"></a>2. Scope**
**This applies to the following:**
<ul><li>
	All sales reps that enter any burning orders.
</li><li>
	Warehouse employees that work with burning operations.
</li></ul>

### **<a name="roles"></a>4. Roles and Responsibilities**
<ul><li>
	<b>Supervisors/Managers:</b> Ensure employees have been properly trained, identify errors, follow up/spot check orders.
</li><li>
	<b>Employees:</b> Identify when to use correct procedure, use proper equipment, double check to make sure everything is correct.
</li></ul>

### **<a name="procedures"></a>5. Procedures**
<ol><li>
	<a name="sales"></a><b>Sales</b>
	<ul><li>
		For all parts except simple shapes (rectangles, baseplates, donuts, etc.), ask customer for CAD files and PDFs.<br> 
		If CAD files are not available, have customer provide legible sketches with <em>all</em> relevant dimensions.<br>
		If a sketch is not available, pass along as much informtaion as possible to the <a href="mailto:processing@pacemakersteel.com">CAM programmers/drafters</a> so a part can be created in CAD then approved by customer.
	</li><li>
		Note any special packaging requirements beyond <a href="ORDER%20ENTRY.md#title">ORDER ENTRY SOP</a> on sales order.
	</li></ul>
<li>
	<a name="warehouse"></a><b>Warehouse</b>
	<ul><li>
		Verify the correct material is pulled (thickness, grade, size, etc.)
	</li><li>
		Verify correct machine parameters (amperage, consumable setups, torch tips, etc.)<br>
		Know the condition of the active consumables. If worn, replace.
	</li><li>
		Run initial part and verify relevant dimensions.<br>
		If parts are out of tolerance or have incorrect dimensions alert one of the the <a href="mailto:processing@pacemakersteel.com">CAM programmers/drafters</a> for corrections.<br>
		Periodically check parts for quality and tolerance.
	</li><li>
		Clean parts as required by customer.
	</li><li>
		Unless otherwise noted, package per <a href="ORDER%20ENTRY.md#title">ORDER ENTRY SOP</a>.<br>
		Apply shipping sticker for identification and tracking.
</li></ol>
